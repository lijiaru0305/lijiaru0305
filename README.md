# Hi, I'm Jiaru

I am an astrophysics postdoc at [CIERA](https://ciera.northwestern.edu/), Northwestern University. My research interests include:
- exoplanetary dynamics
- protoplanetary disks
- dynamical evolution of embedded black holes in AGN disks.
I primarily study theories in these topics using:
- analytical theories
- semi-analytical methods, e.g., numerical ODEs and PDEs
- <i>N</i>-body simulation
- hydrodynamics simulations.
Please check out the [my research page](https://lijiaru0305.github.io/) for more details.
